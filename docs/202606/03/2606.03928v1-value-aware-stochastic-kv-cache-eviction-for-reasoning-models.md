## 摘要
推理模型通过扩展思维链提高了准确性，但其长输出造成了内存和计算瓶颈。KV缓存淘汰方法通过从缓存中淘汰不重要的键值对来降低这一成本，然而与保留完整KV缓存的基于选择的稀疏注意力替代方案相比，这些方法通常导致更低的准确性。我们识别了影响KV缓存淘汰准确性的关键因素。首先，一小部分值状态具有异常大的幅度，淘汰它们会导致灾难性失败，使模型进入重复推理循环。其次，在淘汰过程中引入随机性通过增加缓存多样性提高了准确性。基于这些发现，我们提出了价值感知随机KV缓存淘汰（VaSE），这是一种无需训练的配方，保护了大幅度值状态并促进了多样化的淘汰决策。在六个推理任务上，使用VaSE进行4倍KV缓存压缩的Qwen3模型，在相同稀疏度下取得了比最先进选择方法更高的平均准确率，同时比最强的淘汰方法高出超过4%。总体而言，VaSE弥合了效率与准确性之间的差距，支持FlashAttention2，并为推理模型实现了静态内存占用。

## Abstract
Reasoning models improve accuracy through extended chains of thought, but their long outputs create a memory and compute bottleneck. KV cache eviction methods reduce this cost by evicting unimportant key-value pairs from the cache, yet they often yield worse accuracy than selection-based sparse attention alternatives, which keep the full KV cache. We identify key factors crucial to KV cache eviction accuracy. First, a small fraction of value states have abnormally large magnitudes, and evicting them causes catastrophic failure where models enter repetitive reasoning loops. Second, introducing stochasticity during eviction improves accuracy by increasing cache diversity. Based on these findings, we propose Value-aware Stochastic KV Cache Eviction (VaSE), a training-free recipe that protects large-magnitude value states and promotes diverse eviction decisions. Across six reasoning tasks, Qwen3 models using VaSE with 4x KV cache compression yield higher average accuracies than SOTA selection method at the same sparsity, while outperforming the strongest eviction method by more than 4%. Overall, VaSE bridges the gap between efficiency and accuracy, supporting FlashAttention2 and enabling a static memory footprint for reasoning models.

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：
  1. **价值意识**：保护具有大幅度范围（Range(v)）的值状态，防止它们被淘汰。
  2. **随机性**：淘汰过程中引入概率机制，增加缓存中保留 KV 对的多样性，避免确定性决策导致的模式崩溃。
- **关键技术细节**：
  - **淘汰框架**：采用周期性淘汰，设置持久预算 K 和缓冲区 B（B=64），每 B 步触发淘汰：保留缓冲区中最新的 B 个 token，从剩余 K 个候选 token 中淘汰 B 个。
  - **值状态幅度度量**：使用 Range(v) = max(v_j) - min(v_j) 衡量值状态幅度（与 L2 范数和方差高度相关，0.8+）。
  - **VaSE-AttnV**：
    - 在 SnapKV 基础上扩展：保留 N_v = K/4 个最大 Range(v) 的 token 作为保留集 R_V。
    - 剩余 K - |R| 个槽位通过加权随机采样填充，采样概率正比于平均注意力分数 ¯α_i（式 2）。
    - 随机采样使得每个 token 在所有淘汰步骤中被保留的概率严格为正，而确定性 top-k 会使低于阈值的 token 被永久淘汰。
  - **VaSE-DKV**：
    - 基于 CurDKV 的乘积分数 ℓ(K)_i(G)·ℓ(V)_i(G)（价值项 ℓ(V)_i(G)=||G^T v_i||_2^2 天然放大大幅度值）。
    - 在每个淘汰步骤 t 重新采样高斯投影矩阵 G_t ∈ R^(d×r)（r=20），改变评分标准，避免同一 token 被固定投影反复给低分。
- **公式与算法流程（文字说明）**：
  - 参照图1（上）：缓存包含 K 个持久 token + B 个缓冲区 token，淘汰时对 K 个候选分词进行评分，选择 B 个分数最低的丢弃。
  - VaSE-AttnV：先按 Range(v) 保留 N_v 个 token，然后从剩余候选集中按 ¯α_i 权重不重复采样补满。
  - VaSE-DKV：每个淘汰步骤独立采样高斯投影 G_t，对所有候选 token 计算杠杆分数 ℓ(K)·ℓ(V)，按分数从低到高淘汰 B 个。

## 6. 主要结论与发现
- 大幅度值状态（大 Range(v)）对推理模型至关重要，一旦被淘汰会使模型陷入无休止的重复推理循环。
- 淘汰过程中引入随机性显著提升精度（SnapKV+采样提升 6.6%，CurDKV+采样提升 9.0%）。
- VaSE-AttnV 在 Qwen3-4B 上平均准确率 **59.09%**，超越 SeerAttention-R（58.81%）和最强淘汰方法 R-KV（54.69%）；在 Qwen3-14B 上达 65.81%，同样接近 SeerAttention-R（65.37%）并领先 R-KV（60.90%）4.9%。
- VaSE-DKV 在 CurDKV 基础上带来 7.7%～9.2% 的跳跃式提升。
- 在代码生成中，训练无关淘汰方法（R-KV、VaSE）优于基于学习的选择方法（SeerAttention-R），表明方法泛化性好。
- 大幅度值状态与逐 token KV 缓存量化误差高度相关（相关系数 0.8–0.9），说明该发现对量化也具有重要意义。
- 吞吐量和内存基准表明 VaSE 方法在保持高吞吐的同时实现约 4× 理论压缩比，实际峰值内存低于 R-KV。
