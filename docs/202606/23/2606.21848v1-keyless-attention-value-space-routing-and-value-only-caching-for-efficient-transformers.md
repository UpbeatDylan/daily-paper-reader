## 摘要
我们提出无键注意力（Keyless Attention），一种完全消除键投影的注意力机制，仅操作查询和值。这产生了一个仅值缓存，相比标准注意力，KV缓存内存和访问开销减少恰好50%，同时解码吞吐量匹配或超过标准注意力。除了效率提升，我们引入深度m注意力分解（Depth-$m$ Attention Factorization）：标准注意力计算注意力双线性形式的深度2分解，而无键注意力实现了该系列的深度m实例。当m=3时，无键注意力通过一个值空间路由矩阵匹配标准注意力的投影矩阵数量，该路由矩阵取代了键投影并引入了路由与检索之间的耦合。在五个模型和四种架构（GPT-2 280M、GPT-2 557M、Pythia 410M、Qwen2 1.5B和Llama 3.2 1B）上的实验表明，无键注意力在5个模型中的4个上，其困惑度匹配或优于标准QKV注意力。在下游零样本评估（GPT-2 557M）中，无键注意力在5个常识推理基准中的4个上表现更优，同时全程实现了50%的KV缓存减少。

## Abstract
We propose Keyless Attention, an attention mechanism that eliminates the key projection entirely, operating over queries and values only. This yields a Value-Only Cache that reduces KV cache memory and access overhead by exactly 50% over standard attention, while matching or exceeding standard attention's decode throughput. Beyond efficiency, we introduce Depth-$m$ Attention Factorization: standard attention computes a depth-2 factorization of the attention bilinear form, while Keyless Attention realizes a depth-$m$ instance of this family. At m=3, Keyless Attention matches the projection matrix count of standard attention via a value-space routing matrix that replaces the key projection and introduces a coupling between routing and retrieval. Experiments across five models and four architectures (GPT-2 280M, GPT-2 557M, Pythia 410M, Qwen2 1.5B, and Llama 3.2 1B) show that Keyless Attention matches or outperforms standard QKV attention on perplexity in 4 out of 5 models. On downstream zero-shot evaluation (GPT-2 557M), Keyless Attention outperforms on 4 out of 5 commonsense reasoning benchmarks, while achieving 50% KV cache reduction throughout.

## 1. 核心问题与整体含义（研究动机和背景）
- 标准 Transformer 中的 **Key-Value (KV) 缓存** 是长序列自回归解码的主要瓶颈：随着序列长度和层数线性增长，KV 缓存内存常超过模型参数，且访问带宽受限。
- 现有方法（如量化、剪枝、系统级管理）大多是对 KV 缓存的“事后”优化，而非从根源减轻其负担。
- 本文提出 **Keyless Attention**，彻底消除 Key 投影，仅保留 Query 和 Value，从而将 KV 缓存缩小至仅 Value 缓存（**Value-Only Cache**），内存和访问开销精确减少 **50%**，且无需近似、量化或剪枝。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：标准注意力的注意力分数计算为 $ \text{softmax}(QK^T / \sqrt{d_k}) V $，其中 Key 投影 $W^K$ 专门用于路由，与检索用的 Value 投影 $W^V$ 解耦。本文认为这种分离是冗余的，可以通过 Query 直接与 Value 计算注意力分数来替代。
- **Keyless Attention 公式**：$ \text{Attention}(Q,V) = \text{softmax}\left( \frac{Q V^T}{\sqrt{d_k}} \right) V $，其中 $Q = X W^Q$，$V = X W^V$，Key 投影被移除。
- **理论等价性**（Theorem 1 & 2）：在一定条件（Value 投影满列秩且子空间包含条件）下，标准注意力的注意力分数矩阵 $X W^Q (W^K)^T X^T$ 可被 Keyless Attention 的 $X \tilde{W}^Q (W^V)^T X^T$ 精确匹配，证明 Key 投影在表达能力上是冗余的。
- **深度-m 注意力分解**：将注意力双线性形式 $\Omega = W^Q (W^K)^T$ 或 $\Omega = \tilde{W}^Q (W^V)^T$ 视为 $m$ 个投影矩阵的乘积。标准注意力是深度 2（QKV），而 Keyless Attention 可实例化为深度 3（QVV(3)）：$W^Q = W^{Q_1} W^{Q_2}$，$V = X W^V$，此时投影矩阵数量与标准注意力相同。
- **值空间路由**：注意力路由直接基于 Value 表示，而非独立的 Key 空间，引入路由与检索之间的耦合。这种耦合导致梯度纠缠（$W^{Q_2}$ 与 $W^V$ 相互依赖），起到隐式正则化作用，减少过拟合。
- **推理效率**：预计算合成 $ \tilde{W}^Q = W^{Q_1} W^{Q_2} $，解码时只需一次 Query 投影。Value-Only 缓存仅存储 Value 状态，解码吞吐量超过标准注意力（实验验证）。

## 6. 主要结论与发现
- **性能匹配或超越**：Keyless Attention 在 5 个模型中的 4 个上，最佳验证损失 / 困惑度低于或等于标准注意力；GPT-2 557M 上零样本评测 5 项中 4 项更优（HellaSwag +0.9pt, StoryCloze +1.9pt）。
- **50% KV 缓存减少**：在所有架构上精确实现，且解码吞吐量更高（如 Qwen2-1.5B 上下文 512 时 24.05 vs 22.27 tokens/s）。
- **更好的鲁棒性**：Keyless Attention 在最佳 epoch 后验证损失退化速度更慢，显示出更强的抗过拟合能力（归因于梯度纠缠的隐式正则化）。
- **深度 m 的影响**：QVV(3) 在效率与性能间取得最佳平衡；非线性（ReLU）会破坏性能；随着深度增加收益递减。

## 7. 优点
- **理论贡献**：建立了深度-m 注意力分解框架，统一了从 Luong 注意力（m=1）到标准注意力（m=2）再到无键注意力（m=3）的谱系，为注意力设计提供新维度。

- **强实验验证**：跨多种主流架构（GPT-2、Pythia、Qwen2、Llama）验证，结果一致，表明方法具有通用性。
- **性能与效率双赢**：在多数任务上性能不降反升，同时大幅降低内存和提升吞吐，实用性高。
